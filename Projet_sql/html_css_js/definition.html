L'informateur

    • En informatique, une requête est une interrogation d'une base de données. Elle peut comporter un certain nombre de critères pour préciser la demande. Il existe plusieurs langages de requêtes, qui sont spécifiques à la structure des bases de données. Le plus connu est le SQL, il est exploité dans les bases de données relationnelles (dont les informations sont enregistrées dans des tableaux à deux dimensions).
